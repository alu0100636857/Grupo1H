\section{Objetivos}
\label{1:sec:2}
  El trabajo se ha realizado con los objetivos siguientes:

\begin{itemize}
  \item  Aprender a buscar información adecuada para el desarrollo de la tarea a realizar. Utilizando, entre otras cosas, las herramientas de la BULL.
  \item  Analizar el problema a estudiar y diseñar una solución a dicho problema.
  \item  Crear e implementar un algoritmo de resolución al problema dado mediante Python.
  \item  Verificar mediante pruebas que la solución propuesta es correcta y eficiente.
  \item  Elaborar un informe final sobre el tema utilizando \LaTeX{}
  \item  Realizar una presentación pública del mismo.
\end{itemize}
